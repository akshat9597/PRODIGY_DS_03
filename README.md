# PRODIGY_DS_03

# 🌳 Customer Purchase Prediction using Decision Tree Classifier

## 📌 Project Overview
This project demonstrates building a Decision Tree Classifier using Python on the Bank Marketing dataset.

The objective of this project is to predict whether a customer will purchase a product or service based on demographic and behavioral data. Data preprocessing and exploratory analysis were performed before training the model.

---

## 📊 Visualizations Created

- Count Plot (Customer Purchase Distribution)
- Histogram (Age Distribution)
- Histogram (Campaign Contacts Distribution)
- Heatmap (Feature Correlation)
- Decision Tree Visualization
- Feature Importance Analysis

---

## 🛠 Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

---

## 📁 Dataset

Bank Marketing Dataset containing customer information such as:

- Age
- Job
- Marital Status
- Education
- Default Status
- Housing Loan
- Personal Loan
- Contact Type
- Campaign Contacts
- Previous Campaign Outcome
- Employment Variation Rate
- Consumer Price Index
- Customer Purchase Status (Target Variable)

---

## 🧹 Data Preprocessing Steps

- Loaded dataset using Pandas
- Checked for missing values
- Converted categorical features using Label Encoding
- Split dataset into training and testing sets
- Selected target variable (`y`)
- Prepared features for model training

---

## 🤖 Model Used

**Decision Tree Classifier**

The model was trained to classify whether a customer would subscribe to a product/service based on customer attributes.

---

## 📈 Key Insights

- Customer age and previous campaign interactions influence purchase behavior
- Certain demographic features have stronger predictive impact
- Decision Tree helps visualize decision-making paths clearly
- Model successfully classified customer purchase outcomes
